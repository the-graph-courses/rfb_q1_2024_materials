Introduction

Bienvenue à votre premier devoir d’atelier ! Dans l’exercice de cette semaine, vous appliquerez vos connaissances en R acquises dans les trois leçons préalables de cette semaine :

  • Installation de R et RStudio

  • Comment utiliser RStudio

  • Bases du code R

Trouver et résoudre des erreurs dans le code est une compétence essentielle pour les analystes de données et vous aidera énormément dans votre apprentissage de R.

Le devoir est à rendre le vendredi 19 janvier à 23h59 UTC. Le devoir doit être soumis individuellement, mais vous êtes encouragé à réfléchir en groupe avec des partenaires. Pour soumettre votre travail, veuillez télécharger votre script R sur la page de soumission de l’atelier 1.

Pour commencer, veuillez télécharger le dossier week_01_workshop.zip qui vous a été ENVOYÉ par courriel à partir de trainingteam@thegraphnetwork.org.

Pour démarrer

  1. Localisez le courriel que nous vous avons envoyé avec le dossier de l’atelier en pièce jointe. L’objet est RFB janvier 2024 : Matériel de l’atelier 1.

  2. Téléchargez le dossier zip week_01_workshop.zip -

    Alternativement : vous pouvez télécharger le dossier en cliquant sur le bouton “Télécharger le code de l’atelier” sur la page de l’atelier du sujet WORKSHOP 1 du site du cours.

  3. Décompressez le dossier week_01_workshop.zip. Si vous êtes sur un PC, allez dans votre dossier Téléchargements, faites un clic droit sur le fichier zip téléchargé et cliquez sur “Extraire tout”. Si vous êtes sur un Mac, vous pouvez cliquer directement sur le dossier téléchargé.

  4. Choisissez un dossier de destination où sauvegarder le dossier décompressé. Nous recommandons de créer un nouveau dossier dédié à l’enregistrement du travail lié au R Bootcamp. Vous pouvez également décompresser et sauvegarder les futurs dossiers d’ateliers dans ce dossier.

  5. Dans le dossier décompressé, les instructions de votre exercice sont détaillées dans “week_01_instructions.html” (ce sont les mêmes instructions que celles que vous voyez ici).

  6. Ouvrez le script R exercice_correction_erreurs.R du sous-dossier “scripts” et répondez à chaque question pour compléter le devoir.

  7. Soumettez votre script R sur la page de l’ATELIER 1 sur le site du cours. Le devoir doit être rendu au plus tard le vendredi 19 janvier à 23h59 UTC.
